Gwen’s Redemption is a follow/fight utility bot for Guild Wars 1, built in GwAu3 and inspired by T7TCrusher’s original Follow Bot. Designed for secondary/support accounts, this bot automatically follows a designated party leader, manages heroes, resigns upon entering new explorable areas, uses consumables (PCons), and picks up user-selected loot.

This was my first full GwAu3 project, converted and updated from older GWA2 concepts to work with the modern GwAu3 framework.

Features

✔ Party Leader Follow System

Automatically follows the selected primary account
Adjustable follow positioning / spacing
Town and explorable following support

✔ Automatic Resign Logic

Detects entry into new explorable zones
Automatically resigns and prepares for reset/transition

✔ Automatic PCons Usage

Uses consumables found in the follower account inventory
Helps maintain buffs without manual input

✔ Hero Support

Supports hero-enabled follow accounts
Keeps party synced with the leader

✔ Loot Pickup System

Configurable pickup filters
Pick up selected item types automatically

✔ GwAu3 Updated

Fully updated from legacy GWA2 structure
Built around the current GwAu3 framework
Why Gwen’s Redemption?

There are plenty of follow bots out there already, but this project started as a challenge to learn GwAu3 scripting and build something functional from the ground up. The goal was to create a lightweight, reliable companion bot for multi-account play while learning AutoIt and Guild Wars automation.
